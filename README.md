# Gestion de Contacts

### 1. Prérequis
- Node.js & npm installés.
- Angular CLI installé globalement (`npm install -g @angular/cli`).

### 2. Installation
```bash
# Cloner projet
git clone https://github.com/User-Bird/Mini-Boutique.git
cd gestion-contacts

# Installer les dépendances (node_modules)
npm install

# to run it 
ng serve --host 127.0.0.1 --open
